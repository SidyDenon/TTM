Cette page explique comment demander la suppression de votre compte Tow Truck Mali et des donnees associees.

1) Comment demander la suppression

Envoyez votre demande a l'adresse sidydenon63@gmail.com en indiquant :

  • votre nom complet ;
  • le numero de telephone lie au compte ;
  • le type de compte concerne (client ou operateur) ;
  • la mention explicite : "Suppression de compte".

2) Ce qui est supprime

  • les informations de profil associees au compte ;
  • les donnees de connexion necessaires a l'identification dans l'application ;

3) Ce qui peut etre conserve temporairement

Certaines donnees peuvent etre conservees pendant une duree limitee lorsque cela est necessaire pour des obligations legales, la prevention de fraude, la resolution de litiges ou la tracabilite des missions deja executees.

4) Delai de traitement

Les demandes sont traitees dans un delai raisonnable apres verification de l'identite du demandeur.

5) Demande de suppression partielle

Si vous souhaitez uniquement supprimer certaines donnees sans supprimer le compte, vous pouvez egalement l'indiquer dans votre email.
